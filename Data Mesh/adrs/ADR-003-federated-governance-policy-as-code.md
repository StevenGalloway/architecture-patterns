# ADR-003: Federated governance with policy-as-code gates

## Status
Accepted

## Date
2026-01-11

## Context
Standards must be enforceable without creating central bottlenecks.

## Decision
Implement governance as automated checks:
- naming conventions
- contract validation
- required tests and quality thresholds
- mandatory ownership metadata
- PII tagging and access controls

## Consequences
- consistent standards with autonomy
- requires tooling and shared rules maintained by governance council/platform
